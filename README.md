# Rogue DHCP Detection Lab

This repository documents a DHCP detection engineering project using Splunk and Zeek logs.

The objective of the project was to analyze DHCP lease activity, identify multiple DHCP servers, and investigate potential rogue DHCP behavior within a monitored environment.

##  Technologies Used

- Splunk
- Zeek
- Kali Linux
- VirtualBox

##  Focus Areas

- Detection Engineering
- Threat Hunting
- DHCP Analysis
- SIEM Operations
- Network Visibility
